git status        查看git工作区

git add            添加至暂存区

git commit     提交至本地库

git log             查看版本详细信息

git reflog        查看版本信息

git reset --hard 版本号               版本穿梭



git branch -v          查看分支

git branch 分支名          创建分支

git checkout 分支名       切换分支

git merge 分支名            合并分支到当前分支上

master 分支上修改的内容

hot-fix 分支合并冲突测试



git remote -v                   查看别名

git remote add 别名 远程库地址链接           添加远程库地址并起别名

git push 远程库别名 分支名                          向远程库推送分支

git pull 远程库别名 分支名                          从远程库拉取到分支，更新数据

在网页Github上的更改为了测试 拉取远程库来更新本地库
