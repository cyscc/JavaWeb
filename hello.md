git status        查看git工作区

git add            添加至暂存区

git commit     提交至本地库

git log             查看版本详细信息

git reflog        查看版本信息

git reset --hard 版本号               版本穿梭



git branch -v          查看分支

git branch 分支名          创建分支

git checkout 分支名       切换分支

git merge 分支名            合并分支到当前分支上



master 分支上修改的内容
